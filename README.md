# Vue Progress Circle

Progress Circle Component (Vue 3 + Composition API + SVG)

## Stack

- [Vue 3] — Composition API
- [JavaScript] — component logic
- [Vite] — build tool
- [SVG] — rendering circular progress
- [CSS] — styling without UI frameworks

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:5173
$ npm run dev

# build for production
$ npm run build

# preview production build locally
$ npm run preview
```
